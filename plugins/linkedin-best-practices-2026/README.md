# LinkedIn Best Practices 2026

A Claude Code plugin providing a curated knowledge base for LinkedIn profile optimization, algorithm strategy, and content planning — current as of early 2026.

## Usage

```
/linkedin [task or question]
```

### Examples

```bash
# Load the knowledge base and start working
/linkedin

# Write a headline
/linkedin Write me 3 headline options for a fintech founder

# Review existing profile text
/linkedin Review my About section: "I am a passionate..."

# Look up a specific stat
/linkedin What's the character limit for Experience descriptions?

# Strategy session
/linkedin I post twice a week but get no engagement. What should I change?
```

## What's Inside

- **Algorithm insights** — 2025-2026 shift to Dwell Time, Depth Score, and semantic coherence over engagement farming
- **Section-by-section guidance** — Headline, About (4-part framework), Featured, Experience, Skills, Recommendations
- **Exact specs** — Character limits, image dimensions, search ranking weights, platform limits
- **Content strategy** — Minimum effective approach for low-maintenance, high-authority profiles
- **SEO** — Keyword placement priorities, Google indexing, custom URL strategy

## Components

| Type    | Name                         | Purpose                                  |
| ------- | ---------------------------- | ---------------------------------------- |
| Skill   | linkedin-best-practices-2026 | Full knowledge base (auto-triggers)      |
| Command | /linkedin                    | Explicit context loader for working mode |
