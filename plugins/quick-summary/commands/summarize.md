# Quick Summary

Provide a concise summary of the content the user specifies.

## Behavior

1. If the user provides text or a file path, summarize that content
2. If no content is specified, summarize the current conversation
3. Keep summaries focused and actionable

## Output Format

**Summary:** [2-3 sentence overview]

**Key Points:**

- [Bullet point 1]
- [Bullet point 2]
- [Bullet point 3]

**Action Items:** [Any next steps identified, or "None" if purely informational]
