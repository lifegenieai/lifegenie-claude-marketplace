## Age Descriptors (Detailed)

Age affects vocal texture, energy, and maturity perception.

### Youth Spectrum

- **Adolescent male/female** - Youthful energy, potentially unsteady or
  developing voice
- **Young adult** - Fresh, vibrant, full of energy
- **In their 20s** - Peak vocal clarity, youthful but mature
- **Early 30s** - Established adult voice, confident

### Middle Age

- **Middle-aged man/woman** - Experienced, authoritative
- **In their 40s** - Mature, settled voice
- **Woman in her 50s** - Seasoned, potentially warmer

### Elderly

- **Elderly man/woman** - Weathered, wise, potentially weaker
- **Older woman** - Experienced, potentially grandmotherly
- **Man in his 80s** - Very aged, potentially fragile or raspy

## Tone/Timbre Descriptors (Detailed)

Timbre is the physical quality of the voice, distinct from emotional delivery.

### Depth & Resonance

| Descriptor  | Effect                                           |
| ----------- | ------------------------------------------------ |
| Deep        | Low fundamental frequency, masculine association |
| Low-pitched | Similar to deep, emphasizes pitch                |
| Booming     | Deep with projection, commanding                 |
| Resonant    | Rich, full-bodied sound with harmonics           |

### Texture & Quality

| Descriptor | Effect                         |
| ---------- | ------------------------------ |
| Smooth     | Even, polished, no rough edges |
| Rich       | Full, layered, satisfying      |
| Warm       | Comforting, inviting, rounded  |
| Mellow     | Soft, gentle, not aggressive   |
| Buttery    | Extremely smooth, luxurious    |

### Rough & Edgy

| Descriptor | Effect                            |
| ---------- | --------------------------------- |
| Gravelly   | Rough, stones-in-throat quality   |
| Raspy      | Scratchy, worn, friction in voice |
| Harsh      | Aggressive, unpleasant edge       |
| Throaty    | Deep in the throat, guttural      |

### Airy & Light

| Descriptor | Effect                            |
| ---------- | --------------------------------- |
| Airy       | Breath mixed with voice, ethereal |
| Breathy    | Soft, intimate, whisper-adjacent  |
| Light      | Not heavy, delicate               |
| Thin       | Lacking resonance, reedy          |

### Nasal & Metallic

| Descriptor | Effect                                  |
| ---------- | --------------------------------------- |
| Nasally    | Resonating in nasal cavity, distinctive |
| Shrill     | High, piercing, potentially unpleasant  |
| Tinny      | Thin and metallic, like old speakers    |
| Metallic   | Hard, ringing quality                   |

### Special Effects

| Descriptor | Effect                           |
| ---------- | -------------------------------- |
| Ethereal   | Otherworldly, floating, mystical |
| Robotic    | Mechanical, processed, synthetic |

## Gender Descriptors

Gender influences pitch, vocal weight, and tonal presence. Describe the sound,
not just identity:

### Female Voices

- "A lower-pitched, husky female voice"
- "A feminine voice, light and airy"
- "A powerful female voice with chest resonance"

### Male Voices

- "A masculine male voice, deep and resonant"
- "A softer male voice, gentle and warm"
- "A tenor male voice, bright and clear"

### Neutral/Ambiguous

- "A neutral gender - soft and mid-pitched"
- "An androgynous voice, hard to place"
- "A gender-ambiguous voice with moderate pitch"

## Accent Descriptors

### Prominence Modifiers

| Modifier | Effect                  | Example                         |
| -------- | ----------------------- | ------------------------------- |
| Thick    | Very noticeable, strong | "thick French accent"           |
| Heavy    | Similar to thick        | "heavy Eastern European accent" |
| Slight   | Subtle, hint of         | "slight Southern drawl"         |
| Soft     | Gentle, understated     | "soft Irish lilt"               |
| Crisp    | Clear, precise          | "crisp British accent"          |
| Proper   | Formal, refined         | "proper British accent"         |
| Regional | Local variation         | "regional Australian accent"    |

### Regional Accent Examples

- **British:** "crisp British accent", "thick British accent", "proper British"
- **American:** "neutral American accent", "thick Southern drawl", "New York
  accent"
- **European:** "thick French accent", "German accent", "Eastern European
  accent"
- **Other:** "soft Irish lilt", "Australian accent", "Asian-American accent"

## Pacing Descriptors

### Speed

| Descriptor         | Effect                  |
| ------------------ | ----------------------- |
| Speaking quickly   | Fast delivery           |
| At a fast pace     | Hurried                 |
| At a normal pace   | Standard conversational |
| Speaking normally  | Natural speed           |
| Speaking slowly    | Deliberate, unhurried   |
| With a slow rhythm | Measured, thoughtful    |

### Rhythm & Style

| Descriptor                        | Effect                |
| --------------------------------- | --------------------- |
| Deliberate and measured           | Intentional, precise  |
| Drawn out, savoring each word     | Very slow, expressive |
| Hurried cadence, in a rush        | Fast, urgent          |
| Relaxed and conversational        | Easy, natural         |
| Rhythmic and musical              | Melodic flow          |
| Erratic pacing with abrupt pauses | Unpredictable         |
| Even pacing                       | Consistent timing     |
| Staccato delivery                 | Short, punchy words   |

## Emotion & Attitude

### Negative/Dark

- Sad, emotional, melancholic, angry, menacing

### Personality Traits

- Sweet, gentle, warm, nurturing

## Character/Profession Types

Using profession or character type can quickly establish voice expectations:

### Professions

- TV announcer, radio host, narrator
- Therapist, teacher, professor
- Politician, businessman, executive
- Soldier, drill sergeant, military commander

### Character Types

- Pirate, witch, wizard, goblin, ogre, elf
- Villain, hero, sidekick
- Godlike being, ethereal creature
- Old sage, wise mentor

### Combining Elements

Best results come from combining multiple attributes:

```
"A sarcastic old woman with a thick New York accent, speaking slowly"
```

This combines:

- Emotion: sarcastic
- Age: old woman
- Accent: thick New York
- Pacing: speaking slowly
