# Code Explain

Explain the provided code snippet or file in clear, accessible language.

## Behavior

1. Read the code the user specifies (file path or pasted snippet)
2. Explain what it does at a high level first
3. Break down key sections and their purpose
4. Highlight any patterns, idioms, or notable techniques used

## Output Format

**What This Code Does:** [1-2 sentence plain-language summary]

**How It Works:** [Step-by-step explanation of the logic flow]

**Key Concepts:**

- [Concept 1]: [Brief explanation]
- [Concept 2]: [Brief explanation]

**Things to Note:** [Any gotchas, edge cases, or interesting techniques]
